1- What are Mutation Observers in JavaScript?

MutationObserver is an interface provides the ability to watch for changes being made to the DOM tree.
like when using .observe() in an Element or a NodeList, it observes for changes happening to the selected element or any children under it,
depends on mutations that have been selected (e.g. attributes, childList, ...etc);

2- Explain 3 different CSS selectors.

- Simple selectors (select elements based on TagName, id, class);
- Attribute selectors (select elements based on an attribute or attribute value-rather than ids or classes-);
- Combinator selectors (select elements based on a specific relationship between them);
